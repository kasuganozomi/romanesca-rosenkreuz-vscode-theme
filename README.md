## Installation

1. Navigate to the VS Code Marketplace.
2. Click **Install** to initiate the transmutation.
3. Open the Command Palette → **Preferences: Color Theme** → select **Romanesca Rosenkreuz**.
4. **Mandatory:** Disable native **Bracket Pair Colorization** in your settings to fully experience the custom bracket logic of Rosenkreuz.

*Perfection attained.*
